# lock-cli

> Lock your system from the command-line

Shows the login screen the next time you use the computer.

Supports macOS and Windows.


## Install

With `npm`:
```
$ npm install --global lock-cli
```
Or, with `yarn`:
```
$ yarn global add lock-cli
```


## Usage

```
$ lock
```
